# ✍️ 写作协作 Agent — 结课项目

> 基于「一主三翼双库版」多Agent协作架构的论文写作链路子Agent设计与实现

## 项目概述

本项目设计并实现了一个面向AI专业本科/硕博论文写作场景的**写作协作Agent**，作为多Agent协作系统中的写作链路执行核心。接收主Agent下发的写作子任务，将实验原始数据、分析结果、文献素材转化为规范的论文章节，支持引用自动关联、学术表达润色、多版本迭代管理。

## 架构位置

```
主Agent（全局中枢调度器）
├── 阅读编排器 Agent  ← 文献检索与精读
├── 写作协作 Agent     ← 数据→论文转化（本项目）
├── 个性化推荐 Agent   ← 论文推荐
├── 其他 Specialist ...
└── 双库存储系统
    ├── 库A（持久知识库）— 只读
    └── 库B（任务临时库）— 读写
```

## 文件清单

| 文件 | 说明 |
|---|---|
| `AGENTS.md` | Agent 任务定义：系统定位、任务拆解规则、协作边界、工作原则 |
| `SOUL.md` | 人格规范：语言风格、5大能力准则、约束边界、自检清单 |
| `SKILL.md` | 技能实现：3套标准流水线、JSON调用约定、数据接口、异常处理、质量自检工具 |

## 核心能力

1. **实验写作流水线**：原始实验数据 → 结构化解析 → 引用匹配 → 论文章节生成 → 学术润色
2. **草稿修订流水线**：已有草稿 + 修改意见 → 定位 → 逐条修订 → 全篇校验 → 版本管理
3. **综述生成流水线**：研究主题 → 库A文献检索 → 主题聚类 → 综述正文 → 引用关联
